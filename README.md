# Diwali-sales-analysis
📌 Overview
This project analyzes customer purchasing behavior during the Diwali festival to identify high-value segments and improve marketing strategies.

📊 Key Insights
Women aged 26–35 are the highest contributors to revenue
Top-performing states include Maharashtra, Uttar Pradesh, and Karnataka
Electronics category generates the highest sales

💡 Business Recommendations
Target women (26–35) with personalized campaigns
Focus inventory in high-performing states
Promote electronics for higher revenue

Tools Used
Python, Pandas, Matplotlib
